# 后台默认静态资源

本目录存放 mall-base 后台所使用的默认品牌资源，安装完成后后台会直接使用这些占位图。**上线前必须由设计师提供正式素材并替换同名文件**。

| 文件 | 用途 | 推荐尺寸 | 格式 |
|---|---|---|---|
| logo.png | 后台侧边栏顶部 + 登录页顶部 Logo（统一一张，42×42 展示） | 200×200 | PNG 透明背景 |
| favicon.png | 浏览器标签栏 / 书签 / 桌面快捷方式图标 | 64×64 | PNG/ICO |
| slogan.png | 登录页左侧展示区装饰图（替代 vben 默认 SVG） | 400×400 | PNG 透明背景 |
| avatar-default.png | 管理员/用户未上传头像时的占位头像 | 256×256 | PNG 透明背景 |

## 替换方式

方式一：直接替换同名文件（保持文件名与扩展名不变）。
方式二：登录后台 → 系统设置 → 系统配置 → 基础，上传新图片（自动存到 uploads 目录并更新数据库字段）。

> ⚠ 本目录文件由安装时放入，升级项目时请勿覆盖已被用户替换的图片。如果需要恢复默认，从仓库拉取一次即可。
